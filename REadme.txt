zipdownload.py: downloads the daily zip files from http://www.nemweb.com.au/REPORTS/CURRENT/Yesterdays_Bids_Reports/
and unzips to csv files

zipdownloadARCHIVE.py: downloads the monthly zip files from http://www.nemweb.com.au/REPORTS/ARCHIVE/Yesterdays_Bids_Reports/
and unzips to daily zip files

unzip_zipdownload.py: unzips daily zip files to csv if the
corresponding csv does not exist (use after downloading montly
archives)
